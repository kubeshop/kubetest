# Testkube Version

## **Synopsis**

Shows version and build info.

```
testkube version [flags]
```

## **Options**

```
  -h, --help   help for version
```

## **Options Inherited from Parent Commands**

```
      --analytics-enabled   Enable analytics (default true).

  -v, --verbose             Show additional debug messages.
```
