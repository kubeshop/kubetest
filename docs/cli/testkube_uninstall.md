# Testkube Uninstall

## **Synopsis**

Uninstall the Helm chart registry in the current kubectl context:

```
testkube uninstall [flags]
```

## **Options**

```
  -h, --help          Help for uninstall.
      --name string   Installation name (default "testkube").
```
