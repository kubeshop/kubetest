# Supported Test Types

Testkube supports running a long list of testing tools and frameworks. You can find the list of supported test types below.

If we're not currently supporting a testing framework you need, you can create a [Custom Testkube Executor](../test-types/executor-custom.md) and configure it to run any type of tests that you want. These custom executors can be added to your Testkube installation and/or contributed to our repo.

Testkube also added support for Testkube Container executors, allowing you to run your own container images for executing tests. You can check the guide [here](../test-types/container-executor.md) for more details.
